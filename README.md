# Python-Stack

This is a collecting of multiple python games.

## Projects

- **chopsticks.py** - It is a hand game where players attack and split fingers.
- **cricket.py** - It is a simple super over cricket match simulator with the random outcomes.
- **guessthenumber.py** - It is a simple number guessing game.
- **hangman.py** - It is a simple hangman game with the guessing word being HACKCLUB.
- **horsebetting.py** - It is a horse betting game.
- **lcr.py** - It is a Left Right Center dice game.
- **matchstick.py** - It is a simple matchstick puzzle game.
- **rockpaperscissors.py** - It is a Rock Paper and Scissors game.
- **rpg.py** - It is a survival game where you travel 300 miles while managing health, food, water and ammo
